# meus-projetos-python

Este repositório foi criado para armazenar meus projetos, desafios e exercícios práticos desenvolvidos durante meus estudos diários de Python, mantendo a consistência e aplicando conceitos de lógica de programação.

##  Projetos Cadastrados

### 1. Sistema Acadêmico (`sistema_academico.py`)
* **O que faz:** Um sistema de gerenciamento de notas com menu interativo.
* **Conceitos aplicados:** Funções (`def`), listas dinâmicas (`append` e `pop`), loops de repetição (`while True`) e condicionais (`if/elif/else`).

### 2. Jogo da Adivinhação (`jogo_adivinhacao.py`)
* **O que faz:** Um jogo onde o usuário tenta adivinhar o número secreto gerado pelo computador com limite de tentativas.
* **Conceitos aplicados:** Uso da biblioteca `random`, contadores e controle de fluxo.
